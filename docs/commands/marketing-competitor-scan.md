# /marketing-competitor-scan

Run a competitor scan for Arc90 using public Instagram competitor inputs.

## Required Input
Ask for competitor handles or URLs if none are provided.

## Steps
1. Read `docs/launch-content-pack.md`.
2. Route the task to `arc90-instagram-competitor-research-agent`.
3. Analyze public competitor posts, hooks, CTAs, formats, and visible engagement signals.
4. Return practical changes Arc90 should test next.

## Output
Return:
- Competitors reviewed
- Winning patterns
- Arc90 gaps
- Next hook test
- Next carousel test
- Next TikTok/Reels test
- Things not to copy

## Guardrails
Use public information only. Do not scrape private data, copy creative verbatim, or claim that visible engagement fully explains performance.
